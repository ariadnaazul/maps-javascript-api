Maps Api

NOTAS

Se utiliza la api de Maps Javascript para crear marcadores personalizados.

Además se utiliza la librería Marker Cluster permite reemplazar numerosos marcadores por un cluster que indica la cantidad de marcadores en la zona y permite establecer un icono personalizado.

En este proyecto se pueden encontrar los fragmentos de código para:

  • Inicar marcadores
  • Personalizar marcadores
  • Crear marker clusters
  • Personalizar marker cluster
  • Centrar mapa
  • Desplegar cartel on click
